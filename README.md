# Dockerized Node.js Web Server

Welcome to the Dockerized Node.js Web Server project! This repository contains the source code and Docker configuration files to deploy a simple web server built with Node.js and Express.js, packaged within a Docker container.

## Project Overview

This project serves as a practical guide for understanding how to containerize and deploy Node.js applications using Docker. By following the step-by-step instructions provided in the accompanying Medium blog post, you'll learn how to set up a basic web server, create a Dockerfile, build a Docker image, run a Docker container, and more.

For detailed instructions and explanations, please visit the Medium blog post: [Step by Step guide (Link to Medium)](https://medium.com/@mian.mohsinali343/a-beginners-guide-to-devops-building-a-web-server-using-docker-f9eac7cbd1e2)

## Getting Started

To get started with this project, simply clone the repository and follow the instructions outlined in the Medium blog post.
